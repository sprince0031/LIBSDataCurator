# Tech Stack

## Core Technologies
- **Language:** Java 21
- **Build System:** Maven 3.6+
- **Browser Automation:** Selenium 4.28.1 (for interacting with the NIST LIBS database)

## Libraries & Frameworks
- **CLI Parsing:** Apache Commons CLI 1.4
- **Data Processing:** 
    - Apache Commons CSV 1.10.0
    - org.json 20231013
- **Mathematics & Sampling:**
    - Apache Commons Math 3.6.1
    - Apache Commons RNG 1.6
- **Network & I/O:**
    - Apache HTTP Client 4.5.13
    - Apache Commons IO 2.14.0

## Testing Stack
- **Framework:** JUnit 5 (Jupiter) 5.10.2
- **Mocking:** Mockito 5.11.0

## Infrastructure & CI/CD
- **Version Control:** Git
- **CI/CD:** GitHub Actions (defined in `.github/workflows/`)
