# Synthetic Steel Scrap LIBS (S3-LIBS) Dataset
## v0.9.5

This dataset contains synthetic Laser-Induced Breakdown Spectroscopy (LIBS) spectra of steel scrap samples generated by the LIBS Data Generator tool. 

### sha256 Checksum
```txt
master_dataset.csv - 50172299ce793eaf9e8dac40023e922c6cdf5facd2613cc84d12df38dfb495c5
```

### Reproducing the dataset
To reproduce the dataset on your local machine, follow these steps:
- Download the LIBS Data Generator tool (**v0.9.5**) available in the [v0.9.5 release page](https://github.com/sprince0031/LIBSDataCurator/releases/tag/v0.9.5).
- Download the `instrument_profile.json` configuration file from this directory and place it in the `/conf` directory of the tool.
- Use the `run.sh` script (`run.bat` for Windows) in `/bin` with the following flags to regenerate the dataset:
```bash
  ./run.sh -s -v -n 10 --seed 42
```
- The generated synthetic dataset will be saved in the `/data` directory of the tool as `master_dataset.csv`.
